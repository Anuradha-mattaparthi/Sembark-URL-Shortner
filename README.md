Short Title: Sembark URL Shortner

Requirements

Composer (latest)
PHP 8.4
Node.js & npm
Laravel 12
A database (MySQL / MariaDB recommended)


Local setup

Clone the GitHub repository
Recommended DB name:sembark_url_shortner
npm install
run migrations and seeders: php artisan migrate, php artisan db:seed
Run the frontend build/watch (development):npm run dev
Serve the application locally:php artisan serve


Notes about the app

Authentication: The project uses Laravel Breeze for authentication flows (registration, login, password reset, etc.).
Short URL click tracking: Short URL clicks are processed by jobs (queued background jobs). Make sure queue worker is running when testing real-time click-processing (see Troubleshooting below).
